# AI Usage Description
I used AI as a coding helper while working on this C# shopping cart program. It helped me understand several important programming concepts, especially how classes, arrays, and lists work together in a real system. I was still learning how to properly connect different parts of a program, so AI helped me see how data flows between products, the cart, and order history.
also used AI when I got confused with some of the logic, especially in parts like adding items to the cart, updating quantities, checking stock availability, and handling checkout. It helped me understand how to prevent errors like negative stock, invalid inputs, and crashes when the user enters wrong data. AI also guided me in improving the flow of the program so that each menu option works properly without breaking the system.
One of the most helpful parts was organizing my code into methods like AddToCart, CartMenu, and Checkout. Before, my code was more messy and placed in one big section. AI suggested splitting it into functions so it became cleaner, easier to read, and easier to debug. It also gave ideas on how to properly structure loops, condition checks, and user input validation.
However, I did not just copy everything from AI. I still wrote the code myself and decided how the final program should work. I used AI more as a guide, reference, and learning tool rather than a full solution generator.
